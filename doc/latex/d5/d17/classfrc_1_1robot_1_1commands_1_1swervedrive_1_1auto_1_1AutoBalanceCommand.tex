{\footnotesize\ttfamily boolean frc.\+robot.\+commands.\+swervedrive.\+auto.\+Auto\+Balance\+Command.\+is\+Finished (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

Returns whether this command has finished. Once a command finishes -- indicated by this method returning true -- the scheduler will call its {\ttfamily \doxylink{}{end(boolean)}} method. 

Returning false will result in the command never ending automatically. It may still be cancelled manually or interrupted by another command. Hard coding this command to always return true will result in the command executing once and finishing immediately. It is recommended to use \texorpdfstring{$\ast$}{*} {\ttfamily \doxylink{}{Instant\+Command}} for such an operation.
